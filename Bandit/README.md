# OverTheWire Bandit Writeups

Repository containing writeups and walkthroughs for the OverTheWire Bandit wargame.

Bandit is a Linux and cybersecurity learning platform designed to teach fundamental concepts used in system administration, Linux environments, networking, privilege escalation, scripting, Git usage, and security problem solving.

This repository documents my progression from Bandit Level 0 through Bandit Level 33.

## Repository Structure

```
OverTheWire-Bandit-Writeups/
│
├── Bandit_Levels_0_to_10/
├── Bandit_Levels_11_to_20/
├── Bandit_Levels_21_to_33/
│
└── README.md
```

Each folder contains markdown writeups explaining:

- Objective of the level
- Commands executed
- Investigation process
- Mistakes encountered
- Solution path
- Key takeaways
